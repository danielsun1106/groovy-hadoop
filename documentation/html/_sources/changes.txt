Changes

Version 0.2.0
-------------

.. _`Complete Issue Listing`: https://github.com/tthevis/groovy-hadoop/issues?sort=created&direction=desc&state=closed&page=1&milestone=1

All changes are documented in the `Complete Issue Listing`_.
The main goals for this release were:

* Performance.

  * Use lazy variable injection into groovy scripts
  * Reuse output keys and values 
  * Support `combine` phase
  * `CombineSplits` by default

* Improve documentation and UX

  * Several additional convenience parameters for CLI
  * Better CLI help texts
  * *groovy-hadoop* project page
